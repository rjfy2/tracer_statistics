# tracer_statistics
Statistical analysis of fluorescent tracers

This is the accompanying code to "Statistical analysis of tract-tracing experiments demonstrates a dense, complex cortical network in the mouse", Ypma & Bullmore.


Please reach out to me if any of the code is unclear. Original data can be obtained online from 
-http://connectivity.brain-map.org/ (mouse): by the Allen Institute of Brain Science.
-core-nets.org (macaque): from the series of papers by Markov et al. 
-http://cocomac.g-node.org/services/connectivity_matrix.php (macaque): from the collation work initiated by Prof Kötter, taken over by researchers from the German INCF node and the Juelich Research Institute.
